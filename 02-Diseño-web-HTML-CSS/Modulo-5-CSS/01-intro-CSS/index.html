Maneras de estilizar una etiqueta HTML

  • Estilos en línea:

    NOTA: Los estilos en línea se definen en la etiqueta HTML con el atributo style.

    Estilos en línea

  • Estilos internos:

    NOTA: Los estilos internos se definen en la etiqueta <style> dentro del <head> del documento HTML.

    Estilos internos

  • Estilos externos

    NOTA: Los estilos externos se definen en un archivo CSS separado y se vinculan al documento HTML con la etiqueta <link>.

    Estilos externos

En cuestion de jerarquia de estilos, los estilos en línea tienen la mayor prioridad, seguidos por los estilos internos y los estilos externos. (aunque esto puede variar dependiendo de la especificidad de los selectores y de la colocación del link de los estilos externos) Estilos en línea > Estilos internos > Estilos externos

La lectura de los archivos CSS se hace de arriba hacía abajo, por lo que si se definen dos estilos para un mismo elemento, el que se encuentre más abajo en el archivo será el que se aplique.
La especificidad de los selectores CSS determina cuál estilo se aplica a un elemento en caso de que haya conflictos. La jerarquía de los selectores es la siguiente: ID > Clase > Etiqueta
